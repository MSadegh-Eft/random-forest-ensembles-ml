# Random Forest & Ensembles

**Random Forest** and ensemble methods (Bagging, AdaBoost, Gradient Boosting, Stacking)
for Pima Indians diabetes classification.

Built for a Machine Learning course assignment.

## What this project covers

- EDA and standardized train/test split
- Manual RF hyperparameter search
- Ensemble comparison and stacking classifier

## Project layout

| File | Description |
|------|-------------|
| `main.ipynb` | Ensemble experiments |
| `pima_indians_diabetes.csv` | Dataset |
| `Project_Description.pdf` | Assignment brief |
| `Report.pdf` | Report |

## Quick start

```bash
python -m venv .venv
.venv\\Scripts\\activate          # macOS/Linux: source .venv/bin/activate
pip install -r requirements.txt
jupyter notebook main.ipynb
```

Notebook outputs are saved so plots render on GitHub.
